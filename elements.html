Les 7 Éléments

Dans Genshin Impact, il y a 7 éléments. Les personnages utilisent ces éléments pour combattre et créer des réactions élémentaires.

Tableau des éléments

Réactions élémentaires classiques

  • Vaporisation : Hydro + Pyro (Gros dégâts x1.5 ou x2)
  • Fonte : Pyro + Cryo (Gros dégâts x1.5 ou x2)
  • Gel : Hydro + Cryo (Immobilise l'ennemi)
  • Surchargé : Pyro + Electro (Explosion avec recul)
  • Électrocution : Hydro + Electro (Dégâts continus)
  • Fleurissement : Dendro + Hydro (Crée des noyaux Dendro)
  • Hyperfleurissement : Electro + noyaux Dendro (Très puissant)
  • Dispersion : Anémo + autre élément (Diffuse l'élément)

Réactions lunaires de Nod-Krai (Méta 2026)

Les personnages de Nod-Krai introduisent le système Moonsign qui permet des réactions lunaires beaucoup plus puissantes. Ces réactions dominent la méta en janvier 2026 :

  • Lunar-Bloom : Dendro + Hydro lunaire (Nefer est le meilleur)
  • Lunar-Charged : Electro lunaire (Flins est le meilleur)
  • Lunar-Crystallize : Geo + Hydro lunaire (Zibai sera la meilleure en février)
  • Lunar-Hyperbloom : Version améliorée de l'Hyperfleurissement

Pour activer ces réactions lunaires, il faut avoir un personnage de Nod-Krai dans l'équipe avec le bon Moonsign. Columbina est essentielle pour activer les réactions lunaires car elle fournit l'application Hydro nécessaire.
