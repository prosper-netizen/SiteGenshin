Les Personnages Récents de Nod-Krai

Depuis octobre 2025, Genshin Impact a introduit une nouvelle région appelée Nod-Krai avec des personnages puissants et un nouveau système de combat basé sur les réactions lunaires.

Nouveaux personnages de Version 6.3 (Janvier 2026)

  • Columbina - Élément Hydro, Catalyseur (Fatui Harbinger, sortie le 14 janvier 2026)
  • Zibai - Élément Geo, Épée (Meilleure DPS Geo, sort le 3 février 2026)
  • Illuga - Élément Geo, Lance (Support 4 étoiles pour Zibai)

Personnages de Version 6.2 (Décembre 2025)

  • Durin - Élément Pyro, Épée (Dragon devenu humain, sorti le 3 décembre 2025)
  • Jahoda - Élément Anémo, Arc (Support 4 étoiles, employée de Nefer)

Personnages de Version 6.1 (Octobre 2025)

  • Nefer - Élément Dendro, Catalyseur (Boss du Curatorium of Secrets)

Personnages de Version 6.0 (Septembre 2025)

  • Flins - Élément Electro, Lance (Lightkeeper, DPS Lunar-Charged)
  • Lauma - Élément Dendro, Catalyseur (Support pour Lunar-Bloom)
  • Aino - Élément Cryo, Claymore (Première peronnage jouable de Nod-Krai)
  • Ineffa - Élément Electro, Lance (Sub-DPS Electro)

Meilleurs DPS par élément (Nod-Krai)

  • Meilleur Dendro : Nefer (DPS Lunar-Bloom, sortie octobre 2025)
  • Meilleur Electro : Flins (DPS Lunar-Charged, sortie septembre 2025)
  • Meilleur Pyro : Durin (DPS hybride avec Vaporize, sortie décembre 2025)
  • Meilleur Geo : Zibai (DPS Lunar-Crystallize, sort février 2026)
  • Meilleur Hydro : Columbina (Support pour réactions lunaires, sort janvier 2026)

Système de réactions lunaires

Tous ces personnages de Nod-Krai utilisent le système Moonsign qui permet des réactions lunaires beaucoup plus puissantes que les réactions normales. Pour activer ces réactions, il faut avoir au moins un personnage de Nod-Krai dans l'équipe.
